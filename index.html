Image Processing Pipeline with FastAPI, Celery, and Supabase

In this project, I have developed a robust image processing pipeline utilizing FastAPI, Celery, and Supabase to handle and optimize image data efficiently. The pipeline automates the entire workflow from image retrieval to processing and storage. Initially, images are downloaded from URLs listed in a CSV file. These images are then compressed by 50% to reduce file size without significantly compromising quality. The processed images are uploaded to Supabase, with the original images stored in an 'input' folder and the compressed images in an 'output' folder. The original CSV file is updated with new columns containing the URLs of the compressed images, facilitating easy access and management. The project demonstrates the integration of modern technologies for scalable image processing and storage solutions.
